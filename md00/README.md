# Module 00

## Overview
Introductory set of mini tasks to kickstart programming AVR microcontrollers.

## Toolchain

| Tool | Purpose |
|------|---------|
| `avr-gcc` | Cross-compiler for AVR architecture |
| `avr-objcopy` | Convert compiled output to flashable format |
| `avrdude` | Flash the binary onto the devkit |

## Learning Objectives
1. Getting familiar with the toolchain: `avr-gcc`, `binutils-avr`, `avrdude`
2. Write embedded code in C
3. Debouncing
4. Compile into .hex file
5. Flash program into chip
